SlideShow (fade эффект)

Данный плагин позволяет реализовать слайдшоу с fade-эффектом. У плагина довольно широкий выбор настроек, который позволяет применять его во многих ситуациях: от фейд-ротации обычных изображений до полноценного слайдшоу с авторотацией, паузой на ховер, педжинацией и т.д.

В плагине доступны следующие опции:

				jQuery('div.slideshow').fadeGallery({
					slides: 'ul.slideset > li',          // селектор слайдов которые будут меняться с фейдом
					btnPrev: 'a.btn-prev',               // селектор кнопки "предыдущий слайд"
					btnNext: 'a.btn-next',               // селектор кнопки "следующий слайд"
					activeClass:'active',                // класс активной pagination-ссылки и текущего слайда
					disabledClass:'disabled',            // класс заблокированной кнопки при отключенном циклическом вращении (circularRotation)
					generatePagination: false,           // селектор пустого блока в который сгенерировать pagination-ссылки (по умолчанию отключен - false)
					pagerLinks: '.pagination li',        // селектор существующих pagination-ссылок в HTML (работает вместо generatePagination)
					currentNumber: 'span.current-num',   // селектор блока в котором отображается текущий слайд галлереи
					totalNumber: 'span.total-num',       // селектор блока в котором отображается общее количество слайдов в галлерее
					circularRotation: true,              // циклическая ротация слайдов
					switchSimultaneously: true,          // одновременное переключение слайдов
					disableWhileAnimating: false,        // игнорировать все действия пользователя пока в галерее происходит анимация
					autoRotation: false,                 // авторотация Вкл/Выкл
					pauseOnHover: true,                  // пауза авторотации при ховере блока галереи
					autorotationStopAfterClick: false,   // остановить авторотацию после клика на prev/next/pagination?
					autoHeight: false,                   // автоматическая высота маски галереи под высоту текущего слайда
					switchTime: 4000,                    // задержка авторотации (4000мсек = 4сек)
					animSpeed: 600,                      // длительность фейд анимации (600мсек = 0,6сек)
					useSwipe: false,                     // использовать swipe-жесты для пролистывания слайдов
					disableFadeIE: false                 // отключить fade-эффект в IE (мгновенное переключение)
				});

Требования к верстке

  • Слайды фейд-галлереи должны висеть друг над другом на position:absolute (как стопка бумаги)
  • На активном слайде галлереи z-index должен быть выше, чем на остальных (стилизовать через .active на слайдах)
